# NIWI Test Vectors

These vectors are generated by `lib/niwi/tests/test_vectors.c`.

Regenerate them from the repository root with:

```sh
make -C lib/niwi generate-vectors
```

The vectors are implementation fixtures for encoding, domain-separated
hashing, observable NPRO replay, and fixed-opening commitment scaffolding.
They are not external NIWI, KLP22, Pass, RPBSch, or BIP-340 vectors.
